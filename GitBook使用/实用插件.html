GitBook插件

实用的GitBook插件

插件安装方式

在书籍根目录下创建 book.json文件，在该文件中按照指定格式插入插件对应的代码。

在终端执行命令行

gitbook install

安装插件。

有用的插件

一、导航目录折叠

GitBook默认目录没有折叠效果，导致目录结构过多时，看起来比较费劲，使用下边这个插件可以实现导航目录折叠效果。 使用方式：在 book.json中写入以下内容

 {
     "plugins":["chapter-fold"]
 }

二、code复制代码功能

当文本中有代码时，提供一键复制功能，尤其对于比较长的代码段，非常有用，避免了选中复制的操作。

使用方式：在 book.json中写入以下内容

{
    "plugins":["code"]
}

三、侧边栏宽度可调节

左侧目录结构和右侧文章区域可以拖动调节宽度。

使用方式：在 book.json中写入以下内容

{
    "plugins":["splitter"]
}

四、文章阅读数量统计

记录每个文章阅读的数量。

使用方式：在book.json中写入以下内容

{
    "plugins":["pageview-count"]
}

五、文章页面添加页脚，显示编辑信息。

在文章的底部添加页脚，显示编辑的时间和作者信息。

使用方式：在book.json中写入以下内容

{
    "plugins":["tbfed-pagefooter"],
    "pluginsConfig":{
        "tbfed-pagefooter":{
            "copyright":"Copyright &copy Feng 2020",
            "modify_label": "该文章修订时间：",
            "modify_format": "YYYY-MM-DD HH:mm:ss"
        }
    }
}

六、隐藏GitBook目录下方 Published with GitBook 模块

GitBook默认在目录下方显示 Published with GitBook 按钮。可以设置为隐藏。

使用方式： 在 book.json中写入以下内容

{
    {
    "plugins": [
        "hide-element"
    ],
    "pluginsConfig": {
        "hide-element": {
            "elements": [".gitbook-link"]
        }
    }
}
}
