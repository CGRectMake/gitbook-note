GitBook安装

GitBook的安装

简介

GitBook是使用GitHub/Git 和MarkDown 构建的漂亮书籍的命令行工具。 GitBook可以将您的内容作为网站(html)或者电子书(PDF)输出。

安装方式

通过NPM安装 终端执行命令：

npm install gitbook-cli-g

gitbook-cli-g 是GitBook 的一个命令行工具。它将自动安装所需的GitBook来进行构建。

安装成功后执行下面的命令，查看是否安装成功。

gitbook -V

初始化一本书

初始化

可以通过命令行初始化一本书。终端cd到想要保存书的目录下执行命令：

gitbook init

构建

执行构建命令

gitbook build

构建命令会在目录下生成一个 _book 的目录，内容为静态站点的资源文件。

启动本地服务

使用命令

gitbook serve

会在本地启动一个web服务，可以使用浏览器通过http://localhost:4000/访问
